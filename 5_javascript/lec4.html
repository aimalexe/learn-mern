Lecture: Loops, Control Statements, and Error Handling

do...while Loop

The do...while loop executes the block of code at least once before checking the condition.

for...of Loop

The for...of loop iterates over iterable objects like arrays and strings.

for...in Loop

The for...in loop iterates over object keys or array indices.

Control Statements: break and continue

Control flow statements like break and continue allow you to modify the behavior of loops.

break Statement

continue Statement

Timeouts and Intervals

setTimeout()

Run a function after a specified delay using setTimeout.

setInterval()

Use setInterval to run a function repeatedly at specified intervals.

Exception Handling: try, catch, finally, and throw

try...catch

The try...catch block allows you to catch errors and handle them gracefully.

finally

The finally block always runs, regardless of whether an error occurred or not.

throw Statement

You can create custom errors using the throw statement.

Task for Students

Now it's your turn! Implement the following tasks in JavaScript:

  1. Use a do...while loop to print numbers from 10 to 0.
  2. Create an array of fruits and use a for...of loop to print each fruit.
  3. Create an object with properties for your name, age, and favorite hobby. Use a for...in loop to print both the keys and their values.
  4. Write a for loop that breaks when it reaches 7.
  5. Write a for loop that skips over the number 3 using continue.
  6. Use setTimeout() to log a message after 3 seconds.
  7. Use setInterval() to count from 1 to 5, then stop the interval.
  8. Write a try...catch block that catches a division by zero error.
  9. Create a custom error using throw and catch it in a try...catch block.
